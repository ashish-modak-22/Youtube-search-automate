# 🎥 YouTube Search Automation using Selenium (Python)

## 📌 Project Overview
This project is a **beginner-level Selenium automation project** where a browser automatically opens **YouTube**, types a search query into the search bar, clicks the search button, and displays the search results without any manual interaction.

The main goal of this project is to understand how **web automation works using Selenium**, including browser control, element selection, and user-action simulation.

---

## ⚙️ Technologies Used
- **Python**
- **Selenium WebDriver**
- **Google Chrome**
- **ChromeDriver**
- **VS Code**

---

## 🚀 Features
- Automatically opens the YouTube website
- Locates the YouTube search bar
- Types a search query automatically
- Clicks the search button
- Waits for results and displays the search results page

---

## 🧠 Project Workflow
1. Launch Google Chrome using Selenium WebDriver  
2. Open `https://www.youtube.com`  
3. Locate the search input box using **CSS Selector / XPath**  
4. Automatically type the search keyword  
5. Locate and click the search button  
6. Wait for the results page to load  
7. Search results are displayed successfully
